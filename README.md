## Features
- MVVM architecture using [Observable object](https://github.com/032nnxkitty/SwiftUsefulSnippets/blob/main/Snippets/ObservableObject.swift)
- Core Location
- Geocoder
- Haptic feedback
- Without storyboard
- Without third-party libraries
- UNIT tests
